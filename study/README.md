## 5분 요약 — 이 프로젝트는 무엇인가

**STELLAPET**은 우주 쓰레기를 먹고 자라는 생체 위성(스텔라펫)을 키우는 **다마고치 게임**입니다.

```
알을 품는다 → 부화 → 지상에서 키운다 → 로켓에 태워 발사 → 궤도에서 쓰레기 수거 → 진화
```

기술적으로 특이한 점 세 가지를 먼저 알아두면 코드가 훨씬 잘 읽힙니다.

1. **게임 엔진 라이브러리를 안 씁니다.** Unity도, Phaser도 없습니다. 게임 규칙은 전부 `src/lib/game/engine.ts`에 있는 **평범한 함수**입니다. 그래서 브라우저 없이도 테스트할 수 있습니다.
2. **이미지 파일과 소리 파일이 0개입니다.** 캐릭터 그림은 코드로 그린 픽셀(문자열 배열), 효과음은 코드로 합성한 파형입니다. `public/` 폴더에 png가 몇 개 있지만 그것도 스크립트로 생성한 것입니다.
3. **서버가 거의 없습니다.** 게임 저장은 브라우저 안(localStorage), 서버는 리더보드 하나뿐입니다. 그래서 비행기 모드에서도 게임이 돌아갑니다.

## 개발 환경 준비 (3분)

```bash
# 1. 프로젝트 폴더에서 라이브러리 설치 (처음 한 번)
npm install

# 2. 개발 서버 실행
npm run dev
# → http://localhost:3003 을 브라우저에서 열기
```

코드를 저장하면 브라우저가 자동으로 새로고침됩니다. 끄려면 터미널에서 `Ctrl+C`.

> 💡 이 프로젝트의 개발 포트는 **3003**입니다. 옆 프로젝트(jd-02=3002, jd-04=3004)가 동시에 떠 있을 수 있으니 남의 포트를 끄지 않도록 조심하세요.

**커밋하기 전에는 반드시** 아래를 실행해 빨간 에러가 없는지 확인합니다.

```bash
npm run build   # 문법 검사(린트) + 타입 검사 + 실제 빌드를 한 번에
```
